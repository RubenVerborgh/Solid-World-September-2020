Community Solid Server

September 2020 update

Ruben Verborgh

Ghent University – imec

Community Solid Server is open software
to provide people with their own data pod.

It also will give developers an environment
to create and test new Solid applications.

Community Solid Server has been designed
with a highly modular architecture.

  • Deploy the server in the configuration you need!
  • Want to change functionality?
    • No need to change the server’s code!
    • Just write a module and plug it in.
  • Developers can build their own modules
    and share them with the world.
    • wired together by JSON(-LD) files
